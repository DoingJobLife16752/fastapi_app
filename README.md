# FastApi Application

Тестовый проект, который может показывать данные пользователей и пару базовый эндпоинтоа

## Технологический стек
* **Python**
* **FastAPI**
* **Uvicorn**

## Как запустить проект локально

Через командную строку curl http://localhost:8000 или uvicorn main:app --reload --host localhost

**Клонировать репозиторий:**
   git clone https://github.com/DoingJobLife16752/fastapi_app
